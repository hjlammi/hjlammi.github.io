Technologies

At work I am part of a team working on a customer's video streaming platform. One application I developed for it during my trainee period in the summer 2018 is an integration component between the backend system and YouTube: it is responsible for uploading a video file to YouTube, updating its metadata and finally deleting the video from YouTube. The application was a real learning experience and a fun one as I got to learn some cool new things such as AWS and Terraform.

AWS and Terraform have actually become quite familiar to me during my working career as we are in the process of moving our customer's backend system to the cloud.

Another project I have been working on together with one of our current trainees is a dashboard or an info screen for showing different kind of data. In that project I've worked as a mentor for the trainee as well as worked on mainly the backend implemented with NodeJS. The frontend part of the app is made with ReactJS.

Below I've listed some other projects, school-related or hobby ones, that I've worked on since the beginning of my studies. All of them can be found on my Github page.

  • Javascript
    • A Twitter-style application for writing short messages for logged in users. It was the coursework for Secure Programming course in the spring 2019. The application is deployed to AWS using Codepipeline and it was implemented using NodeJS, Serverless, Terraform, ReactJS and Redux Saga. Test-Driven Development was used in the process: integration testing was done using Cypress.io and Mocha was used for backend testing.
    • Weather application shows the current weather at the user's geolocation (if the geolocation is available, otherwise an error message is shown). Weather data is fetched from OpenWeatherMap's API (https://openweathermap.org/api).
    • DuckApp is a single-page web application I made for Vincit's koodarijahti for summer 2018. With the application you can report duck sightings and view all the reported duck sightings. In the implementation I utilize components provided by Material-UI.
  • Haskell
    • The coursework for Advanced Functional Programming course in the spring 2019. The application is a sort of an text-processing AI from which you can ask simple questions to which it tries to give a logical answer based on the text it was given. It was a really interesting project to work on and I'd love to study funtional programming more in the future! Github page for the coursework.
  • Java
    • LineBreaker asks the user for line width and a piece of text and wraps the text into the lines according to the line width.
    • ASCIIArt gets the name of an ASCII graph text file as a command-line argument and reads its contents into a 2D array. The program allows the user to perform different kinds of commands on the graph, e.g. the graph can be printed on the screen or it can be filtered.
    • SimpleOOPEShell simulates a textual user interface that takes commands from the user and processes files and folders accordingly.
  • THREE.js
    • Christmas greeting application uses THREE.js to render spinning 3D Christmas ball ornaments and falling snow. Click here to view the app's github page.
